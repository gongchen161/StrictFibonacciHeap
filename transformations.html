Transformations

Link

Link node x and y

  • removing x from its original parent, z

  • If x is active, it's attached to the left of y

  • If x is passive, it's attached to the right of y

Active Root Reduction

Relink two active roots x and y or equal rank r

  • Compare x and y, w.l.o.g. assume y < x

  • Link x to y

  • If the rightmost child z of x is passive, link z to root

Root Degree Reduction

Relink three passive linkable nodes x, y, z of the root x and y

  • Assume x, y, z are the 3 rightmost passive linkable children of the root

  • Assume, w.l.o.g x < y < z

  • Make x and y active

  • Link z to y

  • Link y to z

  • Since x is active, make x the leftmost child of the root

One Node Loss Reduction

Relink an active node x with loss 2 to the root x and y

  • Assume x is an active node with loss 2

  • Link x to the root

Two Node Loss Reduction

Relink two active node x, y wwith equal rank r and loss 1

  • Assume there are two active nodes x and y

  • w.l.o.g, x < y

  • Link y to x
