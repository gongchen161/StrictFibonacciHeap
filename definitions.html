Definitions

Rank

  • Number of Active children

  • Only defined for Active/White Nodes

  • Undefined for Passive/Red Nodes

Active Root

  • An Active/White node with Passive/Red parent

  • e.g. Node 2, 7, 8

Loss

  • Number of Active/White children lost

  • Active roots have loss 0

  • Only defined for Active/White Nodes

  • Undefined for Passive/Red Nodes

Linkable Node

  • All children and itself are passive

  • Can only be a Passive/White Nodes

  • e.g. Node 4, 9
