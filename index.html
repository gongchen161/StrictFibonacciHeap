Overview

Strict Fibonacci Heap is the first pointer-based heap implementation that matches the time bounds of Fibonacci heaps in the worst case

The general structure is

  • One heap-ordered tree with N nodes

  • Minimum at root

  • A node can be either an Active/White node or a Passive/Red node

  • All nodes have O(logN) degree

  • All non-root nodes are also maintained in a queue(Q) with position i

  • Root is always passive
