Invariants

For convenience, we define R = 2logN + 6

I1: For an active node, the i-th rightmost active child has rank + loss >= i + 1

Lemma 1: If I1 is satisfied and the total loss is L, then the maximum rank of a nodes <= logN + SQRT(2L) + 2

Proof

Assume x is an active node with the maximum rank r >= k + 1 + logN, where k is the minimum integer such that k(k+1) / 2 >= L. We will use proof by contradiction that x will be a subtree with size N + 1.

We let Tx be the subtree rooted in x. Given I1 is satisfied, i.e. the i-th rightmost active child of x has rank+loss >= i - 1, and the total loss is L. We can notice the minimum subtree that can satisfy this condition is the binomial trees Tb of size 2 ^ r with all nodes being active. Note that we need degree r since x need to have at least r children to have rank r, and we need a binomial tree so that the i-th rightmost active child has rank + loss(0) = i - 1.

Now, we want to prunes the nodes in Tb so that it can have loss L and keep the size as small as possible since we want to prove the contradiction that x is at least size N + 1.

To achieve the above goals, We need to prune the tree L times, and each time we want to cut the grandchildren of the root with the maximum degrees. We know for an binomial tree, there are j grandchildren of degree r - j - 1, so there are Sum 1 … k (j) = k(k+1)/2 grandchildren of degree >= r - k - 1.

Since k(k+1)/2 >= L, We have pruned L times, and no grandchildren of degree < r - k -2 are pruned. That is, the (r-k)-th right-most child w of x has degree r-k-1 > log N and loss 0.

Since it’s a binomial tree, size of Tw is N, and the size of Tx is at least N + 1. Thus, by contradiction, the original statement is true.
